Affix is fixed at the top of the page by default.

You can set offset attribute to change the offset top，the default value is 0.

vue
<template>
  <div>
    this is basic myAffix example.
    <MyAffix :content="content" @update="(val) => (content = val)" />
  </div>
</template>

</script>
